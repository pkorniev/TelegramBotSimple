Simple Telegram bot extension for Yii2.
=======================================
Simple Telegram bot extension for Yii2.

Installation
------------

The preferred way to install this extension is through [composer](http://getcomposer.org/download/).

Either run

```
php composer.phar require --prefer-dist pkorniev/yii2-telegram-bot-simple "*"
```

or add

```
"pkorniev/yii2-telegram-bot-simple": "*"
```

to the require section of your `composer.json` file.
